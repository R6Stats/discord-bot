# R6Stats Discord Bot

This bot is built with [discord.js](https://discord.js.org/#/) to retrieve and display Rainbow Six: Siege stats from [R6Stats](https://r6stats.com)

## Installation
```bash
yarn # or npm install
cp .env.example .env # modify .env to fit needs
```

## Running in Production

```bash
yarn start
```

## Development & Testing

```bash
yarn run dev
```
